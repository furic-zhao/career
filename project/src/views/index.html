    关于本系统

    这是一个基于Framework7构建的WebApp简历。

    使用gulp自动化构建工程（Less/Sass编译、CSS/JS合并压缩、雪碧图自动合成、Imagemin图片压缩等）。

    使用LiveReload作为研发环境、监听文件变动、自动刷新浏览器、多终端自动化测试。
